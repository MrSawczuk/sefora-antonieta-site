QUAL O NOSSO PROCESSO?

1

ORÇAMENTO DO PROJETO

Análise inicial do projeto e elaboração do orçamento detalhado para o desenvolvimento do projeto arquitetônico completo.

2

ENTREVISTA

Etapa onde é determinado o programa de necessidades que norteará o layout dos espaços, a partir das informações fornecidas pelo (a) cliente através de uma entrevista. A coleta de informações poderá ser realizada presencialmente ou online.

3

ESTUDO PRELIMINAR

Compreensão do partido do projeto e dos principais elementos que compõem o mesmo. O estudo traz uma planta para cada pavimento a reformar, contendo informações gerais de cada ambiente, estrutura estimada e descrição dos ambientes com layout. Esta fase é apresentada através de plantas humanizadas.

4

MAQUETE 3D ARQUITETURA

Representações gráficas com a intensão de simular virtualmente o projeto utilizando imagens ultrarrealistas, por onde o cliente compreende em detalhes as soluções desenvolvidas pelo escritório. Através desse estudo o projeto será aprovado e reconhecido.

5

PROJETO EXECUTIVO

Projeto de Arquitetura final, com detalhamentos, especificações técnicas definidas e autorizadas à reforma. O Projeto executivo será constituido a partir de pranchas técnicas, com todas as informações necessárias para compor o orçamento da obra e será definido como manual de execução, assinado pelo Autor. Através dessa etapa, será possível dar início ao desenvolvimento da obra civil ou de reforma.

6

ORÇAMENTO DA OBRA

Preparação do orçamento final detalhado com todos os custos de materiais, mão de obra e serviços da execução.
